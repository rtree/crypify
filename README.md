# crypify

## 🎯 プロジェクト概要

**crypify**は、Shopifyの**Payment Extension**技術を使用し、チェックアウト画面で「クレジットカード」「PayPal」と並んで「Crypto (USDC on Base)」を選択できるようにします。

### 解決する課題
- **Buyer側**: Cryptoを持っていなくても、カードでUSDCを購入して即座に支払い可能
- **Seller側**: 簡単にCrypto決済を自分のストアに統合可能

### 主要機能
✅ Shopify公式チェックアウトへの完全統合  
✅ CDP Smart Wallet (Embedded Wallets)  
✅ Coinbase Onramp (カード→USDC変換)  
✅ Base Chain (L2)での高速・低コスト決済  
✅ 自動注文作成 & 在庫管理連携
